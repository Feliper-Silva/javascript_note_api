# Api clone Evernote

rotas, users e notas definidas na api.

## Tecnologies:

- NodeJs

  - Express
  - JWT
  - BCrypt
  - DotEnv

- Database( Não Relacional)

  - MongoDB

- Editor:
  - VsCode

## inicialização:

Drives:

```bash
yarn
```

or

```bash
npm install
```

Start:

```bash
yarn nodemon
```

```bash
npm run nodemon
```
